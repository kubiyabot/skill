# Screenshots

Screenshots will be added here to showcase the web interface.

To contribute screenshots:
1. Start the web interface: `skill serve --http`
2. Navigate to http://127.0.0.1:3000
3. Take screenshots of key features
4. Save them here with descriptive names

Recommended screenshots:
- dashboard.png - Main dashboard view
- skills-browser.png - Skills listing page
- skill-run.png - Tool execution interface
- service-requirements.png - Service status banner
