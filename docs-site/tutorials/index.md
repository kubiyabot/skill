# Tutorials

Welcome to the Skill tutorials! These step-by-step guides will help you learn how to create and use skills effectively.

## Getting Started

If you're new to Skill, start here:

| Tutorial | Time | Description |
|----------|------|-------------|
| [Your First WASM Skill](./first-wasm-skill) | 15 min | Build a JavaScript skill from scratch |
| [Your First Native Skill](./first-native-skill) | 10 min | Create a CLI wrapper skill with SKILL.md |

## Intermediate

Once you understand the basics:

| Tutorial | Time | Description |
|----------|------|-------------|
| [API Integration](./api-integration) | 20 min | Build a skill that calls external APIs |
| [Testing Skills](./testing-skills) | 15 min | Write tests for your skills |

## What You'll Learn

By completing these tutorials, you'll be able to:

- Create WASM skills using JavaScript/TypeScript
- Create native skills using SKILL.md format
- Handle parameters and configuration
- Call external APIs securely
- Test your skills thoroughly
- Publish and share your skills

## Prerequisites

Before starting, make sure you have:

1. **Skill CLI installed**
   ```bash
   skill --version
   ```

2. **A text editor** (VS Code recommended)

3. **Basic command line knowledge**

## Need Help?

- Check the [Troubleshooting Guide](/guides/troubleshooting)

## Tutorial Format

Each tutorial follows this structure:

1. **Overview** - What you'll build and learn
2. **Prerequisites** - What you need before starting
3. **Step-by-step instructions** - Detailed walkthrough
4. **Expected output** - What success looks like
5. **Next steps** - Where to go from here

Let's get started!
