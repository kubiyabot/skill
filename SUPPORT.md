## Community Support

### GitHub Discussions

For questions, ideas, and community support, use [GitHub Discussions](https://github.com/kubiyabot/skill/discussions):

| Category | Use For |
|----------|---------|
| **Q&A** | Ask questions and get help from the community |
| **Ideas** | Propose new features or improvements |
| **Show and Tell** | Share your skills, projects, and use cases |
| **General** | Chat with the community about anything Skill-related |

### GitHub Issues

For bug reports and feature requests:

- [Report a Bug](https://github.com/kubiyabot/skill/issues/new?template=bug_report.md) - Something isn't working
- [Request a Feature](https://github.com/kubiyabot/skill/issues/new?template=feature_request.md) - Suggest an enhancement

**Before opening an issue:**
1. Search existing issues to avoid duplicates
2. Check the documentation for answers
3. Provide a minimal reproduction case for bugs

## Before Asking for Help

To get the best help, please:

1. **Check the docs** - Your answer might already be documented
2. **Search existing issues** - Someone may have had the same problem
3. **Provide context** - Include version, OS, and relevant logs
4. **Minimal reproduction** - If reporting a bug, provide a minimal example

### Useful Information to Include

```bash
# Get your Skill version
skill --version

# Get system information
uname -a

# Check installed skills
skill list
```

## Security Issues

**Do NOT open public issues for security vulnerabilities.**

Please report security issues via email: [security@kubiya.ai](mailto:security@kubiya.ai)

See [SECURITY.md](SECURITY.md) for our security policy and responsible disclosure process.
